# SoulX-FlashHead Java 版本 - 环境配置指南

## 前置要求检查清单

- [ ] Python 3.10+ (已安装)
- [ ] Java 17+ (需要安装/配置)
- [ ] Node.js 18+ (需要安装/配置)
- [ ] Maven 3.5+ (已安装)
- [ ] FFmpeg (已安装)

---

## 1. 配置 Java 环境

### 检查 Java 是否已安装
```bash
java -version
javac -version
```

### 如果没有安装 Java，安装 JDK 17+

#### Ubuntu/Debian
```bash
sudo apt update
sudo apt install openjdk-17-jdk
```

#### CentOS/RHEL
```bash
sudo yum install java-17-openjdk-devel
```

#### 使用 SDKMAN (推荐)
```bash
curl -s "https://get.sdkman.io" | bash
source "$HOME/.sdkman/bin/sdkman-init.sh"
sdk install java 17.0.10-tem
```

### 配置 JAVA_HOME

#### 查找 Java 安装路径
```bash
update-alternatives --list java
```

#### 设置 JAVA_HOME (临时)
```bash
export JAVA_HOME=/path/to/your/jdk
export PATH=$JAVA_HOME/bin:$PATH
```

#### 设置 JAVA_HOME (永久)
编辑 `~/.bashrc` 或 `~/.zshrc`：
```bash
export JAVA_HOME=/path/to/your/jdk
export PATH=$JAVA_HOME/bin:$PATH
```

然后重新加载配置：
```bash
source ~/.bashrc
```

## 2. 配置 Node.js 环境

### 检查 Node.js 是否已安装
```bash
node -v
npm -v
```

### 如果没有安装 Node.js

#### 使用 NVM (推荐)
```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
source ~/.bashrc
nvm install 18
nvm use 18
```

#### Ubuntu/Debian
```bash
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs
```

## 3. 快速启动方式 (使用 IDE)

如果配置命令行环境比较麻烦，您可以使用 IDE 来运行项目：

### Spring Boot 后端 (IntelliJ IDEA 或 Eclipse)

1. 打开 IDE
2. 导入项目：`SoulX-FlashHead-Java/backend`
3. 等待 Maven 依赖下载完成
4. 找到 `FlashHeadApplication.java`
5. 右键点击 → Run 'FlashHeadApplication'

### Vue 前端 (VS Code 或 WebStorm)

1. 打开 IDE
2. 打开项目：`SoulX-FlashHead-Java/frontend`
3. 在终端中运行：
   ```bash
   npm install
   npm run dev
   ```

## 6. 常见问题

### 问题：JAVA_HOME 未设置
**解决方案**：按照上述步骤 1 配置 JAVA_HOME

### 问题：Maven 依赖下载慢
**解决方案**：配置 Maven 镜像源，编辑 `~/.m2/settings.xml`

### 问题：npm install 慢
**解决方案**：使用淘宝镜像
```bash
npm config set registry https://registry.npmmirror.com
```
